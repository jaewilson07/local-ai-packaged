**June 23: Activity Log & People QuickStart Revisited**

Use the Domo QuickStarts as foundations for measuring and monitoring user engagement, then redevelop the dashboards to track assets and re-implement the dashboard using metrics that capture user engagement as well as trend asset lifecycles.

- Deploy the Activity Log Quickstart.

**June 24: Domo Governance & Instance Health**

Use the Domo Governance Datasets for tracking aging and underutilized assets. Develop workflows for scoring data quality or data profiling. Use the CLI to automate instance cleanup.

- Define your Domo content Archiving Strategy

- Ensure data quality by building data profiling datasets and monitor fluctuations over time

- Automate tracking, monitoring; and archiving aging assets
